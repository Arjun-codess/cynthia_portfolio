
# Cynthia Portfolio ✨

A modern, animated portfolio website inspired by the award-winning Cynthia Ugwu portfolio design. Built with smooth animations, interactive UI elements, and responsive layouts to showcase frontend development skills.

## 🚀 Live Demo

🌐 Live Website: https://arjun-codess.github.io/cynthia_portfolio/

## 📸 Preview

![Portfolio Preview](./preview.png)

## ✨ Features

- Smooth GSAP-powered animations
- Interactive custom cursor effects
- Responsive design for all devices
- Modern minimalist UI
- Animated page transitions
- Hover effects and micro-interactions
- Clean and organized code structure

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll
